# Millionlive::Parser

Parser for "IDOLM@STER MILLION LIVE!" site.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'millionlive-parser', github: 'gomao9/millionlive-parser'
```

And then execute:

    $ bundle
